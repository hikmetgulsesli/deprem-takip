# deprem-takip — Ürün Gereksinim Dokümanı

## 1. Genel Bakış
- **Proje Adı:** deprem-takip
- **Platform:** Web (SPA)
- **Tipoloji:** Canlı veri dashboard uygulaması
- **Özet:** Türkiye ve dünya genelindeki son depremleri interaktif harita üzerinde gösteren uygulama. Kandilli Rasathanesi RSS ve USGS API kaynaklarından veri çeker. Kullanıcılar depremleri büyüklük ve zaman aralığına göre filtreleyebilir.
- **Hedef Kullanıcı:** Deprem bilgilerini takip etmek isteyen Türk kullanıcılar

## 2. Hedef Platform
- **Platform:** Web (mobil uyumlu responsive tasarım)
- **Tarayıcı Desteği:** Chrome, Firefox, Safari, Edge (son 2 versiyon)
- **Hedef Cihaz:** Masaüstü, tablet, mobil

## 3. Fonksiyonel Gereksinimler

### 3.1 Veri Kaynakları
| Kaynak | Açıklama | Endpoint/Format |
|--------|----------|-----------------|
| Kandilli Rasathanesi | Türkiye depremleri | RSS XML feed |
| USGS | Dünya depremleri | GeoJSON API |

### 3.2 Temel Özellikler
| # | Özellik | Açıklama |
|---|---------|----------|
| 1 | Harita Görünümü | Leaflet tabanlı interaktif harita, deprem noktaları |
| 2 | Deprem İşaretçileri | Harita üzerinde deprem konumları, büyüklüğe göre boyut |
| 3 | Büyüklük Filtresi | Minimum büyüklük filtresi (slider, 1-7+) |
| 4 | Zaman Filtresi | Son 24 saat / 7 gün / 30 gün seçenekleri |
| 5 | Deprem Listesi | Tablo görünümünde deprem listesi |
| 6 | Deprem Detayı | İşaretçi tıklandığında detay paneli |
| 7 | Veri Kaynağı Seçimi | Kandilli / USGS / Her ikisi toggle |
| 8 | Otomatik Yenileme | Verilerin periyodik olarak güncellenmesi |
| 9 | İstatistikler | Toplam deprem sayısı, ortalama büyüklük |

### 3.3 Kullanıcı Etkileşimleri
- **Harita tıklama:** Deprem işaretçisine tıklayınca detay paneli açılır
- **Filtre değişikliği:** Harita ve liste anında güncellenir
- **Zoom/Pan:** Harita navigasyonu
- **Tarih seçimi:** Özel tarih aralığı seçimi

## 5. API Entegrasyonları

### 5.1 Kandilli Rasathanesi RSS
- **URL:** Kandilli web sayfası üzerinden RSS/XML
- **Parser:** XML/JSON dönüşümü
- **Veri:** Tarih, saat, enlem, boylam, büyüklük, derinlik, yer

### 5.2 USGS Earthquake API
- **URL:** https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/{magnitude}_{period}.geojson
- **Period:** significant_hourly, significant_daily, 4.5_hourly, 4.5_daily, 2.5_hourly, 2.5_daily, 1.0_hourly, 1.0_daily
- **Veri:** GeoJSON formatında özellikler

## 6. UI/UX Gereksinimleri
- **Tema:** Koyu tema (dark mode)
- **Tasarım Dili:** Modern dashboard, data-driven
- **Renk Paleti:**
  - Arka plan: slate-950
  - Kart/Surface: slate-800
  - Birincil: blue-500 (USGS), orange-500 (Kandilli)
  - Metin: beyaz
- **Tipografi:** Space Grotesk (başlık), Inter (gövde)
- **Layout:** Sol panelde filtreler, ortada harita, sağda isteğe bağlı liste
- **Responsive:** Mobilde filtreler dropdown, harita tam ekran

## 7. Hata Yönetimi
- **API Hatası:** Kullanıcıya Türkçe hata mesajı, yedek veri kaynağı önerisi
- **Ağ Hatası:** Bağlantı yeniden deneme butonu
- **Boş Veri:** "Bu filtre ile deprem bulunamadı" mesajı

## 8. Performans Hedefleri
- İlk yükleme: < 3sn
- Harita render: < 1sn
- Filtre güncelleme: < 500ms
- API yanıt timeout: 10sn
